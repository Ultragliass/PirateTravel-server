# Vacoronation - server

This will be the server side for the Vacoronation project.

## Instructions

- Run script in `sql` folder in your MySql workbench.
- Run `npm i`.
- Alter `sql.ts` with your database details.
- Set `JWT_SECRET=<secret>`.
- Set `PORT=<port>`.
- Run `npm start`.
